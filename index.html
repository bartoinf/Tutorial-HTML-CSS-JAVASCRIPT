TCP -> criador Bob Kahn, objetivo de resolver o problema do NCP de ter que parar a rede para poder fazer a transmissão.
IP -> criador Vint Cerf.
Origem do nome "Internet". A primeira rede com o nome ARPANET foi subdivididadem em MILNET essa parte ficou com os militares, NSFNET a parte científica ficou com as universidades e o restante era rede comerciais. Essas redes precisavam fazer interconecxoes entre elas onde foi chamada de "Interconnect Networking" que de forma abreviada era chamada de "Internetworking", logo após foi abreviada ainda mais para o que já conhecemos a Internet que é uma rede de redes ou um conjunto de muitas redes.
Em 1993 foi criado por Tim Berners-Lee o protocolo "http://", a linguagem HTML e o wwww, World Wide Web conjunto de servidores especializados em http que foca em produzir conteúdos para hipertextos.
